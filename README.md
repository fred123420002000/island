這是根據YouTube這個影片最後一段所描述的問題寫出的小遊戲
https://youtu.be/mLV_vOet0ss?t=956
問題為，如果有一個圓形的島，島外有鱷魚環繞
鱷魚總以4倍於人且離人最近的方向移動，請問島內的人要如何逃出這座島

*在實作此遊戲的過程中學習了
	*八方向鍵盤移動的作法
	*利用畢氏定理使滑鼠追蹤移動更平滑
	*利用三角函數實現圓形移動

PS
問題描述為鱷魚，程式內卻是鯊魚，
是因為曾經在撰寫過程中讓島與鱷魚錯位，導致鱷魚跑到島上，
同事開玩笑說"鱷魚爬上陸地很正常""不然改成鯊魚吧"

PS
原版本只要順著直徑走，鱷魚就會在原處來回彈跳，無法決定該朝哪一邊追逐
最終解決方法為對追逐參考點抖動處理
